## Get started

<Steps>
  <Step title="Clone and install">
    Clone the repository and install dependencies.

    ```bash
    git clone https://github.com/mlaplante/LWD-Invoices.git
    cd LWD-Invoices
    npm install
    ```
  </Step>
  <Step title="Configure environment">
    Copy the example environment file and fill in your credentials for the database, Supabase, and email.

    ```bash
    cp .env.example .env
    ```
  </Step>
  <Step title="Run migrations and start">
    Apply the database schema and launch the development server.

    ```bash
    npx prisma migrate dev
    npm run dev
    ```
  </Step>
  <Step title="Complete onboarding">
    Open [http://localhost:3000](http://localhost:3000), create your account, and follow the guided onboarding flow to set up your organization.
  </Step>
</Steps>

<Tip>
  For production deployments, see the [Self-Hosting Guide](/self-hosting) for full setup instructions including Supabase, email, and payment gateway configuration.
</Tip>
