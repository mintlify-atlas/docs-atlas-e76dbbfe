LWD Invoices (LaPlante Web Development Invoices) is a full-stack, self-hostable invoicing SaaS rebuilt from a legacy PHP application as a modern Next.js application with multi-tenant organization support. It gives you professional invoicing, time tracking, client management, and business operations — all in one place, on your own infrastructure.

## Who it's for

LWD Invoices is designed for:

- **Freelancers** who need to bill clients, track hours, and get paid online without paying monthly SaaS fees.
- **Consultants** managing multiple clients, projects, and recurring engagements.
- **Small businesses** that need multi-user organization support, expense tracking, and financial reporting.

If you want full control over your data and the ability to customize the application, self-hosting LWD Invoices is the right choice.

## Platform capabilities

Beyond the core features, LWD Invoices includes several platform-level capabilities:

- **Multi-tenancy** — Full organization isolation via Supabase Auth. Each user account is tied to one organization with all data strictly scoped.
- **REST API v1** — Bearer-token authenticated API for external integrations covering clients, invoices, and projects.
- **Global search** — Command palette (Cmd+K / Ctrl+K) for quick navigation across invoices, clients, projects, expenses, and tickets.
- **Audit log** — Organization-scoped activity log for all mutations.
- **File attachments** — File uploads via Supabase Storage for invoices, expenses, and proposals.
- **Support tickets** — Internal ticket system with threaded discussions.
- **Mobile responsive** — Optimized layouts across all pages.
